# Portfolio Website

**Period:** May 2025 - Ongoing  
**Type:** Personal Branding Project  
**Repository:** [tharpep/portfolio](https://github.com/tharpep/portfolio)  
**Live Site:** [pryce-tharpe.dev](https://www.pryce-tharpe.dev/)  
**Topics:** Next.js, Personal Website, Portfolio, Tailwind CSS, TypeScript, Vercel

## Technical Stack

**Frontend Framework:**
- **Framework:** Next.js 15.1 (React) with App Router
- **Language:** TypeScript for type safety
- **Styling:** Tailwind CSS + PostCSS for utility-first styling
- **Deployment:** Vercel (automated deployments from GitHub)

**Special Features:**
- **Spotify API Integration:** Real-time music display showing currently playing track
- **Project Showcase:** Dynamic project display with descriptions and links
- **Responsive Design:** Mobile-first responsive layout

## Architecture

**Modern React Patterns:**
- Server-side rendering with Next.js App Router
- TypeScript for type safety throughout
- Component-based architecture
- API routes for external integrations

**Styling & Design:**
- Tailwind CSS utility classes for rapid development
- Responsive design with mobile-first approach
- Modern, clean UI/UX design
- Custom PostCSS configuration

**API Integration:**
- Spotify API integration for real-time music data
- API routes for server-side data fetching
- Automated scripts for Spotify data fetching (`/spotify-scripts`)
- Secure API key management

**Development Workflow:**
- GitHub integration with Vercel for automatic deployments
- Version control with Git
- Continuous deployment on push to main branch
